# antora-demo

## Installation

- Start powershell as admin and run below command to install <b>nvm (Node Version Manager)</b>

```
choco install -y nvm
```

```
npx antora -v
```

```
node -e "fs.writeFileSync('package.json', '{}')"
```

```
npm i -D -E @antora/cli@3.1 @antora/site-generator@3.1
```

```
alias antora='npx antora'
```

```
antora -v
```
